Formazione

Ho completato un percorso di formazione intensivo che mi ha permesso di acquisire competenze solide nei seguenti ambiti:

  • Introduzione al Web Development: Durante il primo periodo, ho imparato i fondamenti del web development, creando le mie prime pagine web con:
    • HTML
    • CSS3
    • Bootstrap
    • Animazioni CSS
  • Fondamenti di Programmazione: Ho sviluppato una web app semplice, imparando:
    • Fondamenti di programmazione (variabili, condizioni, oggetti, cicli...)
    • Gestione delle versioni con Git
    • JavaScript e manipolazione del DOM
  • JavaScript Avanzato: Ho approfondito lo sviluppo con JavaScript, studiando:
    • Design Patterns
    • ES6 e funzionalità avanzate
    • Ajax e Promises per interagire con API esterne
  • ReactJS: Ho imparato a utilizzare React per creare applicazioni scalabili e modulari, lavorando su:
    • Componenti e stato (state)
    • Routing per SPA (Single Page Applications)
    • Gestione delle proprietà (props)
  • Backend Development: Ho acquisito competenze per costruire RESTful APIs e connettere le applicazioni al cloud, usando:
    • Node.js e Express.js
    • MongoDB per il database
    • Middleware e sicurezza delle applicazioni

Inoltre, ho lavorato su progetti settimanali in team per consolidare queste competenze, contribuendo attivamente alla creazione di applicazioni complete.

Obiettivi Professionali

Grazie a una formazione strutturata e a progetti pratici, ho maturato le competenze necessarie per:

  • Creare applicazioni web professionali utilizzando React.
  • Progettare e sviluppare backend scalabili con Node.js, Express e MongoDB.
  • Rilasciare applicazioni sicure e performanti nel cloud.

La mia missione è utilizzare le mie competenze per creare soluzioni tecnologiche innovative che abbiano un impatto reale.
